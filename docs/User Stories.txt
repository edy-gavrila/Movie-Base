User Stories:
1. The user should see a landing page when the page first loads.
2. When the page first loads, the user can use the app as guest or he can authenticate.
3. The user can login, register or use the site in guest mode.
4. When in guest mode all data is saved in localStorage
5. There should be some default movie lists and tv show lists.
6. When viewing movie details, the user can add a movie to favourites list, to watch list, to personal collection list.
